Это папка с файлами движка.
    Здесь лежат все основные файлы, к которым обращается игра во время работы.

active_wall.py 

map.py
    Этот файл отвечает за карту игрового пространства.
    В нем задается положение предметов.


minimap.py
    Этот файл отвечает за отрисовку миникарты.


Music_module.py
    Этот файл отвечает за музыку и мелодии.
    В нем находятся функции и обьект необходимые для взаимодействия с музыкой.


network.py
    Этот файл отвечает за подключение к серверу.


Player.py
    Этот файл отвечает за обьект игрока.
    В нем находится все, что связано с положением игрока и его направлением.
    В нем находятся функции перемещения, проверки на попадание в текстуры
     и на необходимость включения мелодии перемещения.


ray_casting.py
    Этот файл отвечает за нахождение расстояний до стен коридора.
    Это необходимо для отрисовки 3д картинки.
    Так же в этом файле находится функция переводящая обычные координаты в координаты стен.


Sprite.py

texture.py
